Kitchen Coder A delicious way to learn html

Kitchen Coder is a tool to assist beginning students when they are learning the basics of HTML and CSS. It is based on the PRIMM principles and uses the blockly library to present the HTML syntax as jigsaw puzzle pieces. This way students can focus on learning the concepts behind the web elements without having to worry about syntax errors and typo's.


The project was made by the Web and Information Systems Engineering lab at the Vrije Universiteit Brussel.
